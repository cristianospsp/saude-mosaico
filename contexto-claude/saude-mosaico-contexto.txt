PROJETO SAÚDE MOSAICO - CONTEXTO ATUAL

1. STACK TECNOLÓGICA
Backend:
- Java 21
- Spring Boot 3.2.1
- Spring Data JPA
- MySQL 8.0
- Thymeleaf para templates

Frontend (Em desenvolvimento):
- Páginas HTML com Thymeleaf
- CSS puro
- JavaScript básico
- Flatpickr para calendário

Infraestrutura:
- Docker/Docker Compose
- MySQL em container

3. FUNCIONALIDADES IMPLEMENTADAS
- Navegação básica entre telas
- Tela principal com lista de agendamentos (dados mockados)
- Tela de novo agendamento com:
  * Seleção de especialidade
  * Seleção de profissional (dinâmico)
  * Calendário para escolha de data/hora
  * Botões de ação (Agendar/Cancelar)

4. PRÓXIMOS PASSOS

5. COMANDOS IMPORTANTES
# Subir aplicação
cd ~/projetos/saude-mosaico
mvn spring-boot:run

# Subir banco de dados (quando necessário)
docker-compose up -d

# Acessar aplicação
http://localhost:8080/principal
http://localhost:8080/novo-agendamento
